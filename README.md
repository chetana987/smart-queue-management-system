# Smart Queue Management System

A full-stack web application to manage queues efficiently and reduce waiting time using a modern web interface.

## Tech Stack
- Frontend: React.js, JavaScript, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB

## Features
- Queue management system
- Responsive user interface
- REST API integration
- Secure configuration using environment variables

## Setup
1. Clone the repository
2. Install dependencies in frontend and backend
3. Add `.env` files using `.env.example`
4. Run frontend and backend servers
